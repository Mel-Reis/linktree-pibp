
# 🔗 Linktree - Primeira Igreja Batista no Paiol

Este é um projeto de página estilo **Linktree**, desenvolvido para centralizar os principais links da **Primeira Igreja Batista no Paiol**.

## 🚀 Sobre o projeto

O objetivo deste projeto é criar uma página simples, responsiva e funcional para facilitar o acesso às redes sociais da igreja.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 📱 Funcionalidades

* Layout responsivo (adaptado para celular)
* Botões de redirecionamento para redes sociais
* Função de compartilhamento de links
* Interface simples e intuitiva


## 🤝 Projeto voluntário

Este projeto foi desenvolvido de forma **voluntária** para a **Primeira Igreja Batista no Paiol**.

Todo o conteúdo (nome, identidade visual e links) foi utilizado com **autorização da igreja**, exclusivamente para fins **não comerciais**.

## ⚖️ Aviso legal

Este repositório possui caráter **educacional e de portfólio**.

Não há qualquer finalidade lucrativa neste projeto.
Todos os direitos relacionados à identidade da igreja pertencem à instituição.

Caso haja necessidade de alteração ou remoção de qualquer conteúdo, estou disponível para ajustes.
